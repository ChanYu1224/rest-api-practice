# Djangoを使ったREST APIの練習

## Environments
- Ubuntu 18.04 LTS
- Python 3.6.9
- venv
- Django 3.2.7

## Use Case
リポジトリのクローン

```bash=
git clone https://github.com/ChanYu1224/rest_practice.git
```

環境構築

```bash=
Python -m venv django_venv
source django_venv/bin/activate
pip install -r requirements.txt
```

サーバ起動
```bash=
cd products_project/
python manage.py runserver
```
